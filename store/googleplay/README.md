# Google Play 스토어 등록 이미지 (그대로 업로드)

Play Console → 앱 → **스토어 설정 → 기본 스토어 등록정보**에 아래 파일을 그대로 올리면 됩니다.
모든 파일은 Google Play 규격(크기·형식)에 맞춰 제작되어 있습니다.

## ⚠️ 앱 패키지 이름 (applicationId) — 업로드 전 필독
- **패키지 이름: `kr.co.vience.amori`** (iOS 번들 ID와 통일)
- 기존 `com.example.amori`는 Play가 **거부**하는 예시용 이름이라 `kr.co.vience.amori`로 변경 완료했습니다
  (`android/app/build.gradle.kts`, `MainActivity.kt`, `google-services.json`).
- **패키지 이름은 첫 업로드 후 영구 고정 — 절대 변경 불가.** 다른 이름을 원하면 **첫 업로드 전에** 알려주세요.
- **Firebase 후속작업(필수):** Firebase 콘솔에서 Android 앱 `kr.co.vience.amori`를 추가하고
  **실제 `google-services.json`을 내려받아 `android/app/google-services.json`을 교체**하세요.
  (현재 파일은 패키지 이름만 맞춘 상태 — 교체 안 하면 푸시/일부 기능이 실서비스에서 어긋날 수 있음)
- 릴리스 서명키(upload keystore)도 아직 debug 키입니다 — 배포 전 릴리스 keystore를 만들어 `build.gradle.kts`에 연결하세요.

## 이미지 파일

| 업로드 위치 (Play Console) | 파일 | 크기 | 규격 |
|---|---|---|---|
| **앱 아이콘** | `icon-512.png` | 512 × 512 | 32-bit PNG ✔ |
| **그래픽 이미지(Feature graphic)** | `feature-graphic-1024x500.png` | 1024 × 500 | PNG ✔ |
| **휴대전화 스크린샷** (2~8장) | `screenshot-1‥6-*.png` | 1080 × 1920 (9:16) | PNG ✔ |

### 스크린샷 순서(권장)
1. `screenshot-1-hero` — AI가 먼저 소개팅해요 (컨셉/후크)
2. `screenshot-2-persona` — 나를 닮은 AI 페르소나
3. `screenshot-3-auto` — 앱을 닫아도 24시간 가상 소개팅
4. `screenshot-4-report` — 케미 리포트로 미리 확인
5. `screenshot-5-chat` — 매칭 후 진짜 대화
6. `screenshot-6-safety` — 지인 필터·신고·차단 (안전)

> Play는 스크린샷 **최소 2장**을 요구합니다. 6장 모두 올리는 것을 권장합니다.
> 스크린샷은 앱의 실제 화면·기능을 브랜드 목업으로 표현한 것입니다(실제 UI 색상·문구·기능과 일치).

## 데이터 안전(Data safety) — `data-safety.csv`
Play Console → **앱 콘텐츠 → 데이터 보안 → 가져오기(Import)** 에서 `data-safety.csv`를 업로드하면 폼이 자동 채워집니다.
Google 공식 템플릿 형식 그대로이며, amori 실제 처리(수집 12종·**공유 없음**·전송 암호화·계정삭제 지원)를 반영했습니다.
- 선언한 수집 데이터: 이름·이메일·사용자ID·전화번호(해시)·기타개인정보(자기소개/MBTI)·대략적 위치·사진·인앱 메시지·연락처(지인필터 해시)·앱 상호작용·사용자 제작 콘텐츠(페르소나)·기기ID(FCM)
- **공유(제3자)=아니요**: Firebase·LLM(Google)은 위탁 처리자라 Google 정의상 '공유' 아님
- ⚠️ **삭제 요청 URL(필수)**: `PSL_ACCOUNT_DELETION_URL` / `PSL_DATA_DELETION_URL`에 플레이스홀더 `https://vience.co.kr/amori/account-deletion`를 넣어뒀습니다. **실제 삭제 안내 웹페이지를 하나 만들어 그 URL로 교체**하세요(Google이 심사 때 접속 확인함). CSV의 3번째 열만 바꾸면 됩니다.
- `PSL_HAS_OUTSIDE_APP_ACCOUNTS`는 앱 내 계정(이메일/비번)이라 **비워둠**(값 넣으면 import 오류).
- 나중에 **애널리틱스/광고 SDK를 붙이면** 해당 목적·데이터 유형을 추가해야 합니다. **지인 필터를 끄면** '연락처' 항목을 빼세요.
- ⚠️ 이 선언은 **실제 업로드 빌드와 일치**해야 합니다.

## 아직 필요한 것 (이미지 외)
- **512×512 아이콘의 배경 없는 버전**은 불필요합니다(Play는 위 아이콘 하나만 사용).
- 프로모션 **동영상**(선택): YouTube URL. 이미지 아님.
- **태블릿 스크린샷**(선택): 지금은 없어도 등록 가능. 필요 시 요청 주세요(7"/10" 규격으로 제작).

## 스토어 등록 텍스트 (복사·붙여넣기)

**앱 이름 (최대 30자)**
```
amori — AI 프리데이팅
```

**간단한 설명 (최대 80자)**
```
내 AI 에이전트가 먼저 소개팅하고, 나는 미리보기를 보고 결정하는 AI 프리데이팅
```

**자세한 설명 (최대 4000자)**
```
amori는 내 AI 에이전트가 먼저 소개팅을 해주는 새로운 방식의 데이팅 앱입니다.

■ AI가 먼저 만나요
질문에 답하면 나의 성향과 말투까지 학습한 AI 페르소나가 만들어져요.
내 에이전트가 다른 사람의 에이전트와 자동으로 가상 소개팅을 진행합니다.
앱을 닫아도 24시간 대화가 이어져요.

■ 미리보기를 보고 결정해요
에이전트들의 대화를 분석한 '케미 리포트'로 얼마나 잘 맞는지 미리 확인하세요.
잘 맞는 인연만 골라 만남을 신청할 수 있어요.

■ 마음을 열면 진짜 대화
매칭이 되면 직접 채팅으로 대화하고, 약속까지 잡을 수 있어요.

■ 안전하게, 아는 사람은 빼고
· 지인 필터: 주소록의 아는 사람과는 매칭되지 않아요. 연락처는 기기에서 암호화(해시)되며 원문은 서버로 전송되지 않습니다.
· 신고·차단: 부적절한 상대를 즉시 신고하고 차단할 수 있어요.
· 만 19세 이상만 이용할 수 있는 성인 서비스입니다.

지금 amori에서 AI 프리데이팅을 시작해보세요.
```
